🖱️ 点击外部关闭日期选择器

智能的交互体验，点击页面其他地方自动关闭

✨ 新功能特性

智能检测

使用 useRef 和 useEffect 监听全局点击事件， 自动检测是否点击在日期选择器外部。

自动关闭

当检测到点击外部时，自动将 isDatePickerOpen 设置为 false， 提供流畅的用户体验。

性能优化

只在日期选择器打开时添加事件监听器， 关闭时自动清理，避免内存泄漏。

精确判断

使用 contains 方法精确判断点击目标是否在日期选择器内部， 避免误关闭的情况。

🎯 交互测试步骤

  1. 点击日期按钮打开日期选择器
  2. 点击页面其他任何地方（日历区域、按钮、空白处等）
  3. 观察日期选择器是否自动关闭
  4. 再次点击日期按钮，确认可以正常打开
  5. 在日期选择器内部点击，确认不会关闭

🔧 技术实现

使用 React Hooks 实现点击外部关闭功能，确保良好的用户体验和性能。

// 添加 useRef 和 useEffect import React, { useState, useCallback, useRef, useEffect } from 'react' const CalendarHeader = () => { const [isDatePickerOpen, setIsDatePickerOpen] = useState(false) const datePickerRef = useRef<HTMLDivElement>(null) // 处理点击外部关闭日期选择器 useEffect(() => { const handleClickOutside = (event: MouseEvent) => { if ( isDatePickerOpen && datePickerRef.current && !datePickerRef.current.contains(event.target as Node) ) { setIsDatePickerOpen(false) } } if (isDatePickerOpen) { document.addEventListener('mousedown', handleClickOutside) } return () => { document.removeEventListener('mousedown', handleClickOutside) } }, [isDatePickerOpen]) return ( <div> {/* 日期选择器 */} {isDatePickerOpen && ( <div ref={datePickerRef} className="datePicker"> {/* 日期选择器内容 */} </div> )} </div> ) }

💡 Pro Tip: 这个功能使用了标准的 React 模式来实现点击外部关闭。 通过 useRef 获取 DOM 引用，useEffect 管理事件监听器，确保组件卸载时正确清理资源。
